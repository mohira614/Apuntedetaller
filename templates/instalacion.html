Instalación Flask

Flask es un microframework de desarrollo web para Python, que proporciona herramientas básicas y flexibilidad para crear aplicaciones web rápidamente. Se caracteriza por ser ligero y minimalista, lo que permite a los desarrolladores elegir las extensiones y bibliotecas que necesiten para construir sus aplicaciones.

1. Crear un entorno virtual

Antes de instalar Flask, es recomendable crear un entorno virtual para mantener las dependencias del proyecto aisladas del resto del sistema. 

python -m venv venv

2. Activar el entorno virtual

En Windows

venv\Scripts\activate

En macOS / Linux

source venv/bin/activate

3. Instalar Flask

Una vez activado el entorno virtual, instala Flask usando pip: 

pip install flask

Puedes verificar la instalación ejecutando: 

python -c "import flask; print(flask.__version__)"
